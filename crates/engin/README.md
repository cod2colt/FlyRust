# The Engin of Fly Rust

`engin` is the engin of Fly Rust, built with MVVM architecture.

## Features

a MVVM architecture:
- fly_model.rs: fly model
- fly_view.rs: fly view
- fly_viewmodel.rs: fly viewmodel

## Documentation

- API docs: generated via `cargo doc`
- All public items include runnable examples
